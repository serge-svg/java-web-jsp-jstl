# Project created with eclipse

## Implicit objects in JSP
* request - HttpServlet Request
* response - HttpServlet Response
* session - HttpSession
* application -ServletContext
* out - PrintWriter
* config - ServletConfig
* exception - Exception

## JSTL Namespace c tags
* c:out
* c:set
* c:if
* c:choose
* c:forEach
* c:remove
* c:url
* c:param

## JSTL Expresions
* "${a}"
* "${a+b}"
* "${a and b}", "${a && b}"
* "${a or b}",  "${a || b}"
* "${a eq b}",  "${a not eq b}"
* "${a lt b}",  "${a < b}"
* "${a gt b}",  "${a > b}"
